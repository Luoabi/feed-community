# 搜索页面功能说明

## 📱 页面概述

搜索页面提供完整的内容搜索功能，包括搜索历史、热门搜索、搜索建议、搜索结果展示等。

---

## ✨ 核心功能

### 1. 搜索输入
- ✅ 实时输入关键词
- ✅ 自动聚焦输入框
- ✅ 清空关键词按钮
- ✅ 回车键搜索
- ✅ 取消按钮返回

### 2. 搜索建议
- ✅ 输入时显示搜索建议
- ✅ 点击建议直接搜索
- ✅ 本地建议匹配
- ✅ 可扩展为服务端建议

### 3. 搜索历史
- ✅ 自动保存搜索历史
- ✅ 最多保存10条
- ✅ 点击历史快速搜索
- ✅ 删除单条历史
- ✅ 清空全部历史
- ✅ 本地存储持久化

### 4. 热门搜索
- ✅ 展示热门搜索词
- ✅ TOP3特殊标识
- ✅ 点击快速搜索
- ✅ 热门图标动画
- ✅ 可配置热门词列表

### 5. 搜索结果
- ✅ 结果数量统计
- ✅ 有图/无图两种卡片样式
- ✅ 显示标题、摘要、作者
- ✅ 显示浏览量、时间
- ✅ 分页加载更多
- ✅ 点击跳转详情

### 6. 状态处理
- ✅ 加载中状态
- ✅ 空结果提示
- ✅ 没有更多提示
- ✅ 错误提示

## 📡 API接口

### 搜索接口
```javascript
api.searchContent({
  keyword: '关键词',
  page: 1,
  size: 10,
  status: 'published',
  deleted: 0
})
```

**返回数据：**
```javascript
{
  code: 200,
  data: {
    list: [...],
    total: 100,
    page: 1,
    size: 10
  }
}
```

---

## 💾 本地存储

### 搜索历史
- **存储键**: `searchHistory`
- **数据格式**: `['关键词1', '关键词2', ...]`
- **最大数量**: 10条
- **存储方式**: `wx.setStorageSync()`

---

## 🔧 使用方式

### 1. 从首页跳转
```javascript
wx.navigateTo({
  url: '/pages/search/index'
})
```

### 2. 带关键词跳转
```javascript
wx.navigateTo({
  url: '/pages/search/index?keyword=人工智能'
})
```

## 📝 代码说明

### 关键方法

**performSearch()**
- 执行搜索的核心方法
- 保存搜索历史
- 重置分页状态
- 调用搜索接口

**loadSearchResults()**
- 加载搜索结果
- 处理图片URL
- 格式化时间
- 更新页面数据

**saveSearchHistory()**
- 保存搜索历史
- 去重处理
- 限制数量
- 本地存储

**formatTime()**
- 智能时间格式化
- 刚刚/分钟前/小时前
- 天前/日期显示

## 🐛 注意事项

1. **自定义导航栏**
   - 需要适配不同机型的状态栏高度
   - 使用 `navigationStyle: "custom"`

2. **搜索历史**
   - 使用本地存储，注意存储容量
   - 定期清理过期数据

3. **搜索性能**
   - 避免频繁搜索
   - 可以添加防抖处理
   - 缓存搜索结果

4. **搜索建议**
   - 当前是本地匹配
   - 生产环境建议使用服务端接口

---

## 📊 性能优化

1. **防抖处理**
   - 输入时延迟触发搜索建议
   - 避免频繁请求

2. **结果缓存**
   - 缓存已搜索的结果
   - 减少重复请求

3. **图片优化**
   - 使用懒加载
   - 压缩图片质量
   - 使用占位图

4. **分页加载**
   - 按需加载数据
   - 避免一次加载过多
