# 快速测试头像上传功能

## 🚀 快速开始

### 1️⃣ 启动后端
```bash
cd hotaihoduan/BRApi
mvn spring-boot:run
```

### 2️⃣ 打开小程序
1. 打开微信开发者工具
2. 导入项目 `wechatApp`
3. 点击"编译"

### 3️⃣ 登录账号
- 用户名: `admin`
- 密码: `admin123`

### 4️⃣ 测试上传
1. 点击底部"我的"
2. 点击头像或"编辑资料"
3. 点击头像区域
4. 选择一张图片
5. 等待上传完成
6. 点击"保存"

---

## ✅ 检查清单

### 上传前
- [ ] 后端服务已启动

- [ ] 点击头像弹出选择框

### 上传后
- [ ] 头像显示为新图片
- [ ] 点击保存成功

### 持久化
- [ ] 刷新页面头像不变

## 🐛 遇到问题？

### 上传失败
1. 检查后端是否启动
2. 检查是否已登录
3. 查看控制台错误信息

### 头像不显示
1. 检查返回的URL格式
2. 查看 Network 请求
3. 确认静态资源配置

### 保存失败
1. 查看后端日志
2. 检查用户信息格式
3. 确认Token有效
