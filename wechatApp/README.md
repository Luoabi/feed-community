# 火泰内容 - 微信小程序

## 📱 项目概述

这是一个基于微信小程序原生开发的内容社区平台，提供内容浏览、社区互动、内容发布、用户中心等完整功能。用户可以浏览精选内容、发布帖子、参与社区讨论、管理个人内容。

**项目名称**: 火泰内容  
**开发框架**: 微信小程序原生  
**AppID**: wxa3a3b63e1c61dc31  
**后端接口**: BRApi (Spring Boot)

## 🔧 技术实现

### 1. 网络请求封装 (utils/request.js)

**功能特性:**

```javascript
const BASE_URL = 'http://localhost:8080/api'
// 真机调试时改为局域网IP
```

**请求方法:**

- `request.post(url, data)` - POST请求

- `request.delete(url, data)` - DELETE请求

### 2. 图片处理 (utils/image.js)

**功能:**
- ✅ 图片URL处理

**处理类型:**
- 封面图片 (cover)
- 用户头像 (avatar)

### 3. 全局状态管理 (app.js)

**全局数据:**
```javascript
globalData: {
  userInfo: null,    // 用户信息
  isLogin: false,    // 登录状态
  token: null        // 认证Token
}
```

**全局方法:**
- `checkLogin()` - 检查登录状态

### 4. 分页加载

**实现方式:**
- 下拉刷新：重置page=1，重新加载

- 加载状态：loading标志防止重复请求
- 加载完成：hasMore标志判断是否还有数据

```bash
git clone <repository-url>
cd wechatApp
```

## 🐛 常见问题

### Q1: 真机调试时无法连接后端？
**A:** 
1. 确保手机和电脑在同一局域网
2. 将 `BASE_URL` 改为电脑的局域网IP
3. 关闭电脑防火墙或添加端口例外
4. 在开发者工具中勾选"不校验合法域名"

### Q2: 图片无法显示？
**A:**
1. 检查图片URL是否正确
2. 确保图片服务器可访问
3. 检查是否配置了图片域名白名单
4. 使用默认占位图

### Q3: 登录后刷新页面需要重新登录？
**A:**
- Token已存储在本地，检查 `app.js` 中的 `checkLogin()` 方法
- 确保 `wx.getStorageSync('token')` 能正确获取Token

### Q4: 上传图片/视频失败？
**A:**
- 当前版本暂未实现服务器上传
- 需要后端提供上传接口
- 实现 `uploadImages()` 和 `uploadVideo()` 方法

### Q5: 发布内容后看不到？
**A:**
1. 检查发布接口是否成功
2. 刷新列表页面
3. 检查内容状态是否为已发布

### Q6: 评论无法发送？
**A:**
1. 确保已登录
2. 检查评论内容不为空
3. 检查评论接口是否正常

### Q7: 分页加载不生效？
**A:**
1. 检查 `hasMore` 标志
2. 检查 `loading` 状态
3. 确保后端返回正确的分页数据
