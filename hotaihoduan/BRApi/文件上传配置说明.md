# 文件上传配置说明

## 配置文件位置
`src/main/resources/application.yml`

## 上传路径配置

### 方式1：使用相对路径（推荐开发环境）
```yaml
file:
  upload:
    path: uploads
    domain: http://localhost:8080/api
```
- 文件会保存到项目根目录的 `uploads` 文件夹
- 例如：`C:/Users/Abb/Desktop/zwjBS/hotaihoduan/BRApi/uploads/`

### 方式2：使用绝对路径（推荐生产环境）
```yaml
file:
  upload:
    path: D:/uploads
    domain: http://your-domain.com/api
```
- 文件会保存到指定的绝对路径
- Windows示例：`D:/uploads/` 或 `D:\\uploads\\`
- Linux示例：`/var/www/uploads/`

## 文件存储结构
```
uploads/
├── images/           # 图片文件
│   └── 2026/
│       └── 04/
│           └── 12/
│               └── xxx.jpg
└── videos/           # 视频文件
    └── 2026/
        └── 04/
            └── 12/
                └── xxx.mp4
```

## 访问URL格式
- 图片：`http://localhost:8080/api/uploads/images/2026/04/12/xxx.jpg`
- 视频：`http://localhost:8080/api/uploads/videos/2026/04/12/xxx.mp4`

## 文件大小限制
- 图片：最大 5MB
- 视频：最大 50MB

可在 `application.yml` 中修改：
```yaml
spring:
  servlet:
    multipart:
      max-file-size: 50MB
      max-request-size: 100MB
```

## 注意事项
1. 确保上传目录有写入权限
2. 生产环境建议使用绝对路径
3. 建议定期清理过期文件
4. 可以考虑使用OSS（阿里云、腾讯云等）存储文件
